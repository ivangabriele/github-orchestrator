# Github Orchestrator

[![img-license]][lnk-license] [![img-npm]][lnk-npm] [![img-github]][lnk-github]

Orchestrate your Github repositories and harmonize their setup files from your CLI.

**⚠️ Work In Progress!**

## Install

```sh
npm i -g github-orchestrator
```

## Usage

```sh
gor [command]
```
